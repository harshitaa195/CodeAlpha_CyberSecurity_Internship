Secure Coding Review

Description:

This project is developed as part of the CodeAlpha Cyber Security Internship.

The project reviews a simple login application to identify common security vulnerabilities and provides recommendations for secure coding practices.

Security issues Identified:

* Hardcoded credentials
* Plain text Password
* Weak authentication
* No Account Lockout and no multi-Factor authentication

 Recommendations::

* Password Hashing
* Secure db storage
* Multi-Factor Authentication
* Strong Password policy
