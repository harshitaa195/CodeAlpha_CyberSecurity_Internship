     Phishing Awareness Training

Description

This project is created as part of the CodeAlpha Cyber Security Internship.

It is an awareness presentation designed to educate users about phishing attacks, fake emails, fake websites, social engineering techniques, and methods to prevent phishing.

 Topics covered:

* What is Phishing?
* Types of phishing
* Fake email identification
* Fake Websites identification
* Social engineering
* Prevention tips
* Quiz

Technologies Used
