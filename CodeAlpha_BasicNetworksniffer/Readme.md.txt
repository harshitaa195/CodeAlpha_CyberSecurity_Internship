 Basic Network Sniffer

Description

This project is developed as part of the CodeAlpha Cyber Security Internship

The Basic Network Sniffer captures and analyzes network packets using Python and Scapy. It displays the source IP address, destination IP address, and protocol information to help understand how network communication works.

Features:

* Captures network packets
* Displays source IP Address
* Displays destination IP Address
* Displays protocol information


Technologies Used

* Python
* Scapy
